About

This Web application grabs top videos from Reddit and makes a YouTube playlist (like this one). It can be convenient if you're in the mood for some entertainment, but don't feel like clicking on each individual video.

Frequently Asked Questions

  1. Why is creating a YouTube playlist so slow? The application uses the YouTube API to create playlists. There are limits on how often an application can call the API. Once these limits are reached, the application must wait for a while before it can make more API calls. This waiting is what causes the delay.
  2. Is my account information safe? It is as safe as any of the other data you are keeping with Google. Once you give your permission, this application uses OAuth 2.0 to access YouTube on your behalf, but never has access to your password. You can also revoke the the permission at any time.
  3. Why aren't the most recent Reddit videos showing? In order to reduce the burden on Reddit, this application keeps a local cache of the top 100 entries. This cache is updated every 30 minutes or so.
  4. Why are some videos being skipped? Besides YouTube, Reddit links to videos on various sites such as Vimeo, Dailymotion, and others. Since YouTube playlists can only contain YouTube videos, links to sites other than YouTube can only be skipped.
  5. Where is the source code for the application? It is being kept on github.
